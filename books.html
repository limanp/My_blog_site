This classic book is a comprehensive source of information about the C# programming language and related infrastructure. In the 8th edition of the book, you'll find functionality in the latest versions of C# 7.0 and 7.1 and .NET 4.7, as well as brand new chapters on the lightweight Microsoft .NET Core cross-platform infrastructure, including .NET Core 2.0. The book covers ASP.NET Core, Entity Framework (EF) Core, and more. along with the latest updates to the .NET platform, including those made to Windows Presentation Foundation (WPF), Windows Communication Foundation (WCF), and ASP.NET MVC.
